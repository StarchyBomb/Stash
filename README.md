# 📥 Stash

**เก็บทุกอย่างไว้ก่อน กลับมาเปิดทีหลัง — ปุ่มเดียวจบ**

Stash มีสองเวอร์ชัน ใช้แยกกันได้:

| | Browser Extension | Desktop App |
|---|---|---|
| **เก็บอะไร** | แท็บเบราว์เซอร์ทั้งหมด | แอปเดสก์ท็อปที่เปิดอยู่ |
| **เปิดคืน** | เปิดแท็บกลับในหน้าต่างเดิม | เปิดแอปทั้งหมดกลับมา |
| **เลือกได้** | เก็บทุกแท็บ | ✅ เลือกเฉพาะแอปที่ต้องการ |
| **ปิดคืน** | — | ✅ กดปิดแอปในเซสชันซ้ำได้ |
| **ไอคอน** | — | ✅ แสดงไอคอนของแต่ละแอป |
| **Stack** | Manifest V3 · vanilla JS | Tauri v2 · Rust · HTML/CSS/JS |
| **รองรับ** | Chrome / Edge | Windows / macOS |

---

## 🌐 Stash — Browser Extension

เก็บแท็บทั้งหมดในหน้าต่างไว้เป็นเซสชัน แล้วเปิดคืนทีหลังในหน้าต่างเดิม

### ติดตั้ง

1. เปิด `chrome://extensions` (หรือ `edge://extensions`)
2. เปิด **Developer mode**
3. กด **Load unpacked** → เลือกโฟลเดอร์ `stash/`

---

## 🖥️ Stash Desktop

เก็บแอปเดสก์ท็อปที่เปิดอยู่ทั้งหมด (หรือเลือกเฉพาะบางตัว) แล้วเปิดคืนทีหลังได้ด้วยคลิกเดียว

### ฟีเจอร์

- **เลือกแอปได้** — ติ๊กเลือกเฉพาะแอปที่ต้องการเก็บ
- **ไอคอนแอป** — แสดงไอคอนจริงของแต่ละโปรแกรม
- **เปิดคืน / ปิดคืน** — เปิดแอปกลับมา แล้วกดปิดซ้ำได้ไม่จำกัด
- **System tray** — ปิดหน้าต่างแล้วยังทำงานต่อเบื้องหลัง
- **Autostart** — ตั้งเปิดพร้อม Windows/macOS ได้

### Dev

```powershell
cd stash-desktop
npm install
npm run dev        # รันโหมดพัฒนา
npm run build      # สร้าง .exe installer
```

### ต้องมี

- [Node.js](https://nodejs.org/) ≥ 18
- [Rust](https://rustup.rs/)
