Why Now

หน้าต่างเวลาเพิ่งเปิด: ambient AI ที่เป็นส่วนตัว

01

โมเดลเล็กลงเครื่องได้

wake word, speech-to-text และการจัดกลุ่ม context รันบนเครื่องได้จริงแล้ว — เร็ว ฟรี ออฟไลน์ ไม่ต้องส่งขึ้นคลาวด์

02

วิกฤตสมาธิ + hybrid work

คนทำงานจากหลายที่ หลายอุปกรณ์ หลายเครื่องมือ ความเจ็บปวดเรื่อง context-switch พุ่งสูงและเกิดซ้ำทุกวัน

03

ตลาดกลัว AI สอดส่อง

Rewind/Recall จุดกระแสต้าน — เกิดดีมานด์ชัดสำหรับ AI ที่ local-first และไม่แอบอัดทุกอย่าง

คลื่นทั้งสามมาบรรจบพอดี — และยังไม่มีใครครองพื้นที่นี้

The Solution

Stash — ผู้ช่วยงานที่รันบนเครื่องคุณ

จับ

Context Engine

รู้ว่าคุณเปิดแอปและแท็บอะไร โฟกัสอันไหนนานสุด — จากทั้งเดสก์ท็อปและเบราว์เซอร์ ในเครื่องล้วน

เรียก

Jarvis Mode

พูดหรือพิมพ์คำสั่ง แล้วเปิดคืน workspace ทั้งชุด สร้าง/เคลียร์เซสชัน — ในคำสั่งเดียว

จำ

Work Memory

"เมื่อวานทำอะไรค้าง?" — AI สรุป context ตอนคุณกลับมา ให้เข้างานต่อได้ใน 10 วินาที ไม่ใช่ 20 นาที

เสียงพาเข้า — ความจำรั้งไว้

Live Today

ไม่ใช่แค่ไอเดีย — มีของจริงที่ใช้ได้แล้ว

  • Chrome Extension — เก็บแท็บทั้งหน้าต่างเป็นเซสชัน เปิดคืนได้ (2 ภาษา)
  • Desktop App (Tauri + Rust) — เก็บ/เปิดคืนโปรแกรมพร้อมไอคอนจริง, tray, autostart
  • Landing + demo เล่นได้จริง — จำลอง macOS ให้ลอง stash/restore ในเว็บ
  • Open source (AGPL-3.0) — v0.2.1 ปล่อยแล้ว, มี release + ผู้ใช้จริงเริ่มโหลด

Defensibility

Moat 01

สะพาน 2 ฝั่ง

Moat 02

On-device / privacy

รันในเครื่อง ไม่แอบอัดจอ = ความไว้ใจ ที่เป็นทั้งจุดขายและกำแพงกันคู่แข่งที่พึ่งคลาวด์

Moat 03

Data flywheel

ยิ่งใช้ ยิ่งรู้จักวิธีทำงานของคุณ → ผู้ช่วยฉลาดขึ้น → ย้ายหนียากขึ้น
