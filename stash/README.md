# 📥 Stash

**เก็บแท็บทั้งหมดไว้ก่อน กลับมาเปิดทีหลัง** — browser extension ปุ่มเดียวจบ สำหรับ Chrome / Edge

## ฟีเจอร์เดียวที่มี

1. กดไอคอน Stash → **"เก็บแท็บทั้งหมด แล้วปิด"** → ทุกแท็บในหน้าต่างถูกเซฟเป็นเซสชัน (ตั้งชื่อได้) แล้วปิดให้หมด
2. กลับมาเมื่อไหร่ → กด **"เปิดคืน"** → ทุกแท็บเด้งกลับมาในหน้าต่างใหม่

- เก็บเฉพาะแท็บ `http/https/file` (หน้า chrome:// เปิดคืนไม่ได้อยู่แล้ว)
- ข้อมูลอยู่ใน `chrome.storage.local` ในเครื่องล้วน ไม่มี cloud ไม่มี account

## ติดตั้ง (โหมดนักพัฒนา)

1. เปิด `chrome://extensions` (หรือ `edge://extensions`)
2. เปิด **Developer mode**
3. กด **Load unpacked** → เลือกโฟลเดอร์ `stash` นี้

## Stack

Manifest V3 · vanilla JS · ไม่มี build step — โฟลเดอร์นี้คือตัว extension เลย
